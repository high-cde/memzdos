# ZDOS · NEURO · OS

Sistema neuro-computazionale evolutivo basato su:
- Ingest neurale (FlyWire-like)
- Grafo neurale
- Quantum fingerprint
- Routing neurale
- DSN-CORE wrapper
- DSN-LIVE backend + v2 event stream
- Neuro-IO layer
- NeuroDynamics engine
- Quantum State layer
- Visualizer

Struttura:
- ingest/        → dataset + ingest
- router/        → grafo, router, quantum
- dsn_core/      → DSN wrapper
- dsn_live/      → backend + event stream
- neuro_io/      → event bus
- neuro_dynamics/→ dinamica neurale
- quantum_state/ → stato quantico
- visualizer/    → vista logica


## Evoluzione Neuro-Computazionale
